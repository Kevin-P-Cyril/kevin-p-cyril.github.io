# 📊 Data Analyst Portfolio — Kevin P Cyril

A sophisticated, editorial-style portfolio designed to showcase data analytics skills, projects, and professional experience. This portfolio uses a **config-driven architecture**, allowing for instant content updates without deep-diving into HTML code.

## 🌟 Key Features

* [cite_start]**Config-Driven Content:** Update your entire bio, projects, and contact info in one JS object[cite: 1, 11].
* [cite_start]**Built-in Editor:** An on-page "Admin Panel" to preview content changes in real-time[cite: 1].
* [cite_start]**Clean Typography:** Uses *Cormorant Garamond* for an elegant, premium look[cite: 1].
* [cite_start]**Responsive Design:** Fully optimized for mobile, tablet, and desktop viewing[cite: 1].
* [cite_start]**Recruiter Focused:** Direct links to PDF resumes, GitHub repositories, and LinkedIn[cite: 1, 9].

## 🚀 Quick Setup (Windows)

1.  [cite_start]**Prepare Assets:** * Place your resume at `assets/Kevin_P_Cyril_Resume.pdf`[cite: 1, 3, 13].
    * [cite_start]Place your headshot at `assets/kevin.jpg`[cite: 1, 5, 13].
2.  [cite_start]**Run Setup:** Double-click `setup.bat`[cite: 1, 7]. [cite_start]This will verify your folders and open the site automatically[cite: 1, 7].

## 🛠️ Customization Guide

[cite_start]To change the content, open `index.html` in any text editor and find the `DEFAULT_CONFIG` section[cite: 1, 11].

### 1. Personal Details
[cite_start]Modify your name, title, and location to reflect your professional identity[cite: 1, 11].

### 2. Contact Information
[cite_start]Update the `contact` section with your professional email and social links[cite: 1, 11]:
* [cite_start]**Email:** Opens the default mail client via `mailto:`[cite: 1, 9, 11].
* [cite_start]**Phone:** Formatted for mobile dialers via `tel:`[cite: 1, 9, 11].

### 3. Projects & Experience
[cite_start]Add new projects to the `projects: [ ]` array[cite: 1, 12]. You can customize:
* [cite_start]**Project Title & Description**[cite: 1].
* [cite_start]**Feature Bullets:** Highlight specific achievements[cite: 1].
* [cite_start]**Tech Stack:** List the tools used (e.g., Python, SQL, Power BI)[cite: 1].
* [cite_start]**Accent Color:** Choose a unique color for each project card's side-bar[cite: 1].

---

## 📂 File Structure

* [cite_start]`index.html`: The core portfolio engine and content configuration[cite: 1, 13].
* [cite_start]`setup.bat`: Windows utility for easy local launching[cite: 1, 13].
* [cite_start]`assets/`: Folder for your resume PDF and profile images[cite: 1, 2, 13].
